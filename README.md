# Calculadora MCP Básica

Este es un MCP server en Python listo para desplegar en **Render**.

## Instrucciones

1. `pip install -r requirements.txt`
2. `python server.py`
3. El servidor MCP estará disponible automáticamente por HTTP/HTTPS.

Funciones disponibles:

- **sumar(a, b)** — Retorna la suma de a + b
- **restar(a, b)** — Retorna la resta de a – b
